# 🤖 AI Resume Analyzer & Interview Preparation Assistant

## 📌 Overview

AI Resume Analyzer & Interview Preparation Assistant is a full-stack web application that helps job seekers evaluate and improve their resumes based on a given job description. The application leverages Generative AI to analyze resumes, provide ATS-style feedback, generate personalized interview questions, and optimize resumes for better job alignment.

---

## 🎯 Problem Statement

Many candidates submit resumes that are not tailored to specific job descriptions, reducing their chances of being shortlisted.

This project aims to solve this problem by:

* Analyzing resumes against a job description.
* Providing AI-powered resume feedback and scoring.
* Generating personalized technical and behavioral interview questions.
* Optimizing resumes to better match job requirements.

---

## 🚀 Features

* User Authentication (JWT)
* Resume Upload (PDF)
* Job Description Analysis
* AI Resume Scoring
* Resume Optimization
* Technical & Behavioral Interview Question Generation
* Download Optimized Resume

## 📂 Project Workflow

1. User registers and logs in.
2. Uploads resume and job description.
3. AI analyzes resume relevance.
4. Resume score and improvement suggestions are generated.
5. Technical and behavioral interview questions are created.
6. AI optimizes the resume.
7. User downloads the optimized resume.
